البرمجة

تعريف البرمجة
-
كتابة بعض البيانات ( حروف - أرقام - علامات ) وفق قواعد معينة لتمر بعملية المعالجة لتنتج معلومات ( مواقع - برامج - أنظمة تشغيل - إلخ )
مزايا البرمجة
-
  1. توفير الوقت
  2. السهولة
  3. السرعة
  4. التصميم المريح
عيوب البرمجة
-
  1. ضرر الصحة بسبب كثرة الجلوس أمام الكمبيوتر
  2. الالتزام بالبيانات والقواعد المعطاة في لغة البرمجة
  3. أنها تلتزم بالبيانات والقواعد المعطاة فقط
حلول عيوب البرمجة
-
  1. تنظيم الوقت في الجلوس أمام الكمبيوتر
  2. تنزيل محرر يصحح الأكواد تلقائياً
مجالات البرمجة
-
  1. مجال الويب مثل (facebook) وينقسم هذا المجال إلى قسمين
    • Front-end
    • Back-end
  2. مجال تطبيقات الهاتف مثل (برنامج الآلة الحاسبة)
  3. مجال تطبيقات الكمبيوتر مثل (برنامج الفوتوشوب)
  4. مجال أنظمة التشغيل مثل (ويندوز وماك وأندرويد)
  5. مجال الألعاب مثل (PUBG)
  6. مجال قواعد البيانات
أقسام لغات البرمجة
-
  1. على حسب المستوى :-
    • لغات منخفضة المستوى : الأقرب إلى لغة الآلة والأبعد عن فهم الإنسان ( Assembly language )
    • لغات عالية لمستوى : الأبعد عن لغة الآلة والأقرب إلى فهم الإنسان ( Python - Ruby - Java )
  2. على حسب طريقة التنفيذ :-
    • لغات البرمجة المترجمة : يتم فيها استخدام برنامج المترجم كي يحول اللغة إلى لغة الآلة ( C )
    • لغات البرمجة المُفسّرة : يتم فيها استخدام برنامج المفسر كي يحول اللغة إلى لغة الآلة ( Python )
  3. على حسب الاستخدام :-
      على حسب المجال :-
    • مجال الويب مثل (facebook) وينقسم هذا المجال إلى قسمين
      • Front-end
      • Back-end
    • مجال تطبيقات الهاتف مثل (برنامج الآلة الحاسبة)
    • مجال تطبيقات الكمبيوتر مثل (برنامج الفوتوشوب)
    • مجال أنظمة التشغيل مثل (ويندوز وماك وأندرويد)
    • مجال الألعاب مثل (PUBG)
    • مجال قواعد البيانات
